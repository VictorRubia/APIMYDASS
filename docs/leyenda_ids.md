# Leyendas de la BBDD

## Modelo de datos Sensors

### device_type

- 0: Dispositivo móvil
- 1: WearOS Derecha (TicWatch)
- 2: Empática E4
- 3: WearOS Izquierda (TicWatch)

### sensor_type

- 1: Acelerómetro
- 5: Luminosidad
- 6: Presión
- 8: Proximidad
- 10: Aceleración lineal
- 12: Humedad relativa
- 13: Temperatura ambiente relativa
- 18: Detector de pasos
- 19: Cuentapasos
- 21: Ritmo cardíaco
- 2411: GPS
